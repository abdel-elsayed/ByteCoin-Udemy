## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `Lottie` by adding it to your `Podfile`:

```ruby
platform :ios, '13.0'
use_frameworks!
pod 'Lottie'
```

To get the full benefits import `Lottie` wherever you import UIKit

``` swift
import UIKit
import Lottie
```
